CSS

  • Three kinds of CSS styling: inline, internal CSS style sheet and external CSS style sheet
  • Inline - style an element directly in the HTML file
  • A margin indicates how much we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Inline CSS - styling an element directly in an HTML file (applying CSS to a single element, say, a heading will be written in red)
  • Internal CSS - styling multiple elements directly in an HTML file (applying CSS to multiple elements, say, ALL headings will be in red)
  • External CSS - most common, CSS is implemented with separate style sheet, all styling is coded in separate file
  • The "*" symbol is a "wildcard" symbol; it means that the declarations following it are applied to ALL visible elements on the page (.e.g. * {margin:0 ; padding: 0; })

Git/GitBash Cheatsheet

  • cd - go back one directory
  • cd [directory name] - nav to specific directory
  • git status - which branch am I in currently?
  • pwd - which directory am I in currently?
  • code . - open VS code within current directory
  • git checkout - navigate to different branch
  • git checkout -b [name of new branch] - create a new branch and nav to it
  • git diff --check - check for whitespace errors
  • git add -A - add changed files in working directory to staging area of git (ready to commit)
  • git commit -m [include message] - commit changed files to repo with summary message included
  • touch style.css - Create CSS file named "style.css" within current directory
