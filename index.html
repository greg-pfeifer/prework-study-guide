GitBash Command Cheatsheet (Running)

  • cd - go back one directory
  • cd [directory name] - nav to specific directory
  • git status - which branch am I in currently?
  • pwd - which directory am I in currently?
  • code . - open VS code within current directory
  • git checkout - navigate to different branch
  • git checkout -b [name of new branch] - create a new branch and nav to it
  • git diff --check - check for whitespace errors
  • git add -A - add changed files in working directory to staging area of git (ready to commit)
  • git commit -m [include message] - commit changed files to repo with summary message included
